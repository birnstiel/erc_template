ERC Starting Grant Template

by Tilman Birnstiel

This repository contains my own personal template for an ERC starting grant proposal, based on the 2015 call for proposals. Included are:

  • the template tex-files for Part B1 and Part B2 of the proposal including some auxiliary files (bibliography files, header, ...).
  • the template commitment letter for the host institution.
  • a python script that builds the publication list based on NASA ADS bibcodes (so it will work mainly for astronomers). To make it work, edit the script to include the bib codes for all the papers you want to list, put in the last names of yourself (in order to bold-face your name) and of your PhD advisor (to highlight which publications are with or without the advisor) and finally also add the bibcodes of the papers you want to be highlighted.
  • A Makefile that should compile everything and provides some more functionality like translating the summary to a plain txt file and counting words.
